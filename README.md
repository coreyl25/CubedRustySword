Rusty Sword Cubed Edition is a 3D action-adventure RPG that follows the redemption journey of Russell, a retired knight who must overcome his past failures to save Princess Alaia from the villainous Rantor the Goblin King.
This project is a complete reimagining of the original 2D side-scrolling platformer, now featuring a top-down/third-person 3D perspective with expanded combat mechanics and exploration.
